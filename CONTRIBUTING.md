# Contributing

This is a personal PM portfolio repo — not a traditional open-source project — but feedback is welcome.

## How to engage

- **Questions about a case study?** Open a [Discussion](https://github.com/asifalhye/pm-portfolio/discussions) (Q&A category).
- **Spotted a typo or broken link?** Open an issue or small PR.
- **Hiring / collaboration?** Email ahye@wayfair.com or connect via LinkedIn (link in profile README).

## Ground rules

- Please don't open drive-by PRs that rewrite case studies — context matters and these docs mirror real work narratives.
- Sanitized data only; no proprietary Wayfair metrics in issues or PRs.

Thanks for reading.
